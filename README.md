# Signal

Signal is a personal goal and task tracker web app that helps users manage tasks and goals with a modern, calendar-first interface. The app focuses on task management, goal management, progress tracking, and an interactive calendar view.

## Key Features
- Task management: add, edit, delete, view, and search tasks
- Goal management: yearly resolutions, monthly goals, daily goals, and custom categories
- Progress tracking: dashboards with charts and percentages for today, week, month, and year
- Calendar view: interactive calendar with task scheduling and date-based task browsing
- UI/UX: dark theme, soft accents, responsive layout, subtle animations

## Tech Stack
- React + TypeScript
- Vite
