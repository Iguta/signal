# Testing Guide

## Commands
Run from `signal/codebase`:

```bash
npm test
npm run build
```

## Coverage (PRD Must-Have Features)
- Task management: create tasks via the Tasks page.
- Task search: global search filters tasks across views.
- Goal management: add monthly goals and view lists by timeframe.
- Progress tracking: dashboard shows progress for today, week, month, and year.
- Calendar view: add tasks in the calendar and see them on the selected day.
